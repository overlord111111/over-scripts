# 📜 Over Scripts

Coleção de scripts úteis do dia a dia — automação, scraping, bypass e utilitários diversos.

## Scripts

| Script | Descrição |
|--------|-----------|
| `scraper.py` | Web scraping genérico com fallback |
| `bypass.py` | Bypass de restrições comuns |
| `watcher.sh` | Monitoramento de arquivos/diretórios |
| `cleaner.py` | Limpeza e organização |
| `recon.sh` | Recon básico automatizado |

## Uso

```bash
git clone https://github.com/overlord111111/over-scripts.git
cd over-scripts
python scraper.py --help
```
